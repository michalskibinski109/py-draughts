American checkers

class draughts.boards.american.Board(starting_position: ndarray | None = None, turn: Color | None = None)[source]
Board for American checkers.

Game rules:

  • Board size: 8x8

  • Short moves only

  • Only the king can capture backwards

  • Capture - choose any

GAME_TYPE = 23

PDN game type. See PDN specification.

STARTING_COLOR = -1

Starting color. Color.WHITE or Color.BLACK.
